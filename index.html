Williams Api

Kalkylator

Kan räkna på tre olika räknesätt

• Plus

• Minus

• Gånger

Nödvändiga Parametrar

• nr1

• nr2

• typ

Observera att parametern för gånger är = ganger

Exempel URL :

http://localhost:8080/kalkylator?nr1=500&nr2=500&typ=ganger

Du får tillbaka svaret i jason format

Img

Returnarar en bild slumpmässigt av två möjliga bilder

URL :

http://localhost:8080/img

Sedan returnaras en bild

csv fil

Returnarar en csv fil i jason format

URL :

http://localhost:8080/showcsvfile

Sedan returnaras csv filen

Stensaxpåse

Du får upp ett formular och du väljer mellan sten, sax eller påse, och sedan trycker du på spela

URL :

http://localhost:8080/stensaxpåseform

Exempel svar: Ditt val = sten, datorns val = påse. Datorn vann

För att får resultat

URL :

http://localhost:8080/stensaxpåseresultat

Du får all statistik returnerad i jason

Exempel svar: { "games":"4", "wins":"2", "losses":"1", "ties": "1"}
